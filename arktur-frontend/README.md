# Project setup

### Install Node

https://nodejs.org

### Clone Repository

```
git clone https://github.com/darafsa/fmi-lehrplanung.git
```

or

Use GitHub Desktop:

<img src=".\assets\git-clone.png" style="zoom: 30%;" />

### Install Modules

Navigate to Project and run

```
cd ./fmi-lehrplanung
```

```
npm install
```

### Serve Project

```
npm run serve
```

Your Project will be served at http://localhost:8080/
